# 숙제 데이터 작성 가이드 — 가경T용

> 핵심 한 줄: **문항마다 `영역`과 `유형(type)`만 정확히 적어주시면, 나머지 진단·코멘트는 시스템이 만듭니다.**
> 유형을 구체적으로 쓸수록("관형사형 전성어미"처럼) 코멘트가 콕 집어 생생해집니다.

---

## 1. 가경T가 채워주실 것 — 이 표만 주시면 됩니다

주차마다 아래 표 형태로 주세요. (한글/엑셀/그냥 텍스트 다 OK)

| 번호 | 정답 | 배점 | 영역 | 유형(개념) | 틀리면 보이는 약점 | 보완 방향 |
|----|----|----|------|----------|----------------|---------|
| 1 | 3 | 2 | 문법 | 형태소 분석 | 어근/접사 구분을 놓침 | 단어를 직접 쪼개 표시하며 풀기 |
| 2 | 5 | 3 | 문법 | 관형사형 전성어미 | ‘-ㄴ/-ㄹ’ 기능을 시제로만 봄 | 어미의 기능을 문장에서 확인 |
| … | | | | | | |

- **필수**: `번호`, `영역`, `유형`  → 채점·약점진단·코멘트가 전부 여기서 나옵니다.
- **있으면 좋음**: `정답`, `배점`(기본 2점, 변별문항 3점), `약점`, `보완 방향`, 전국 `오답률(%)`
- **주차 머리 정보**: 강좌명 / 몇 주차 / 그 주차 영역(예: "1주차 · 문법 — 단어와 형태소")

그리고 학습 진단 설문(시간/방식/복습/집중)은 모든 주차 공통이라 **그대로 재사용**합니다. 바꾸고 싶을 때만 말씀 주세요.

---

## 2. 그러면 제가 이렇게 바꿔드립니다 (자동)

위 표 → `data/hw-<강좌>-w<주차>.json` 으로 변환. 예시(2번 문항):

```json
{ "no":2, "answer":5, "three":true, "wr":51, "area":"문법", "type":"관형사형 전성어미",
  "coreElement":"관형사형 전성어미 ‘-ㄴ/-ㄹ’의 문법적 기능 파악",
  "weakness1":"전성어미를 시제 표지로만 이해함",
  "remediation":"어미가 문장에서 하는 역할(관형어 만들기)을 확인하며 풀기" }
```

그리고 약점 패턴(`patternRules`)·강점(`strengthRules`)도 표를 보고 제가 묶어드립니다.
예: 2·5번이 모두 ‘어미’ 계열이면 → "어미 기능 약점" 패턴으로 묶어 진단.

---

## 3. 이 값들이 코멘트에서 이렇게 쓰입니다

| 입력 필드 | 코멘트에서 하는 일 |
|---------|------------------|
| `area`(영역) | "이번 **문법** 파트가 헷갈리죠?" 처럼 공감/맥락 |
| `type`(유형) ⭐ | "**관형사형 전성어미**가 아직 헷갈리는 것 같아요" — 약점 콕 집기 |
| 정답률(자동 계산) | 우수/양호/노력필요 어조 분기 + "92%! 잘 잡고 있어요~" |
| `strengthRules` | "**문장 성분** 쪽은 든든해요!" — 강점 인정 |
| 학습 진단 설문 | "막히면 바로 답지 보지 말고 5분만 더 버텨봐요~"(해설 의존형 등) |

> 즉, **코멘트의 ‘콕 집기’ 정확도 = `type`을 얼마나 구체적으로 적느냐**에 달려 있습니다.
> "문학 일반" 보다 "화자의 정서", "표현법의 기능"처럼 적어주세요.

## 4. 완성되는 코멘트 예시 (실제 생성물)

> 서연 학생, 이번 주도 과제 잘 챙겨왔네요!
>
> 다른 건 잘 풀었는데, 관형사형 전성어미 · 형태소 분석 부분은 조금 더 단단히 잡아야 할 것 같아요.
>
> 이번 주는 관형사형 전성어미에 중점을 두고 복습해보면 좋을 것 같아요~ 여기 한 번 제대로 잡아두면 뒤에 나오는 것들이 훨씬 수월해져요~ 막히면 바로 답지 보지 말고 5분만 더 버텨보는 연습도 같이 해봐요~
>
> 다음 주 과제도 빠짐없이! 약속~

조교는 이 초안을 보고 **한두 군데만 손질 → [복사]/[문자로 보내기]** 하면 끝.
같은 학생도 **[다른 표현으로]** 누르면 문장이 바뀌고, 학생마다 약점이 달라 **친구끼리 겹치지 않습니다.**

## 5. 앞으로 (자동 분석)

교재 숙제 범위(PDF·이미지·문제 텍스트)와 정답을 주시면, 위 1번 표(번호·영역·유형·약점·보완)를
**제가 직접 분석해서 초안으로 채워** 드립니다. 가경T는 검수·수정만 하시면 됩니다.
- 문학: 화자/정서, 표현법, 시상 전개, 서술상 특징, 시점 등 유형 자동 분류
- 문법: 형태소·단어 형성·음운 변동·문장 성분·높임·시제 등
- 독서: 정보 구조, 추론, 〈보기〉 적용 등

처음 한두 주차는 함께 맞춰보며 유형 명칭·약점 표현을 가경T 스타일로 튜닝하면, 이후엔 거의 그대로 쓰실 수 있습니다.
